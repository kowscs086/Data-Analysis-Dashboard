# Pizza-Sales-Data-Analysis (Interactive Dashboard using MS Excel)

## Project Objective
The objective of this project is to analyze pizza sales data to understand sales performance, customer preferences, and revenue trends. This analysis helps the business identify key insights that can be used to improve sales and make data-driven decisions.

## Dataset Used
- Pizza Sales Excel Dataset
- <a href="https://github.com/kowscs086/Data-Analysis-Dashboard/blob/main/pizza_sales_excel_file.csv">pizza sales data</a>
  

## Questions (KPIs)
- What is the total revenue generated?
- Which pizza category generates the highest sales?
- Which pizza size is ordered the most?
- What are the peak order days and months?
- Which pizza type has the highest quantity sold?
- How do sales vary by category and size?
- What is the average order value?
- Which pizzas contribute the most to total revenue?

## Process
- Data cleaning and preprocessing in Excel  
- Created pivot tables for analysis  
- Designed interactive dashboards using charts and slicers  
- Visualized key metrics for better understanding
